PROJECT OVERVIEW

This dashboard tracks the NY Fed Global Supply Chain Pressure Index (GSCPI) — a composite measure of disruptions in global supply chains, published monthly. A reading above 0 signals above-average pressure; April 2026 jumped to +1.82 σ, the largest single-month increase since the post-Ukraine shock of March 2022.

The analysis decomposes that jump using a proxy model built entirely from public data — no access to the NY Fed's proprietary BCI survey data. 22 indicator blocks (freight costs, price surveys, port flows, chokepoint vessel counts) are z-scored and combined via PCA + ridge regression to attribute the observed change to its component sources.

Key result: The 4-model ensemble explains +0.66 pp (58%) of the +1.144 pp observed change. Top contributors: Census BTOS delivery survey (+0.43 pp), ISM Supplier Deliveries (+0.28 pp), PPI Transport (+0.28 pp). The residual likely reflects BCI survey data not available publicly.

METHODOLOGY
How the model works

The GSCPI Proxy Model replicates the NY Fed's Global Supply Chain Pressure Index using only publicly available data, without access to the proprietary BCI survey data the NY Fed uses.

Step 1 — Block construction: ~22 indicator blocks are built from FRED, BLS, Census, PortWatch, ISM, and other sources. Each block is z-scored vs its full available history. Blocks span freight costs, price surveys, physical port flows, and chokepoint vessel counts.

Step 2 — PCA proxy: All block z-scores are combined via Principal Component Analysis. PC1 explains ~31% of variance and correlates at r = 0.75 with the official GSCPI level (full sample). The PC1 score is linearly rescaled to GSCPI units for display.

Step 3 — Attribution: Ridge regression (α = 1–5 depending on window) is fit on 4 training windows to predict monthly GSCPI changes. The 4-model ensemble mean decomposes the April 2026 +1.144 pp change. Ensemble mean: +0.66 pp (58%).

Step 4 — PortWatch integration: Daily vessel counts from the IMF PortWatch ArcGIS API for 7 global chokepoints, aggregated to monthly means. Pressure z-scores computed vs 2019–2026 baseline.

Training window rationale

Four training windows are used. The ensemble approach averages over window-specific mis-specifications:

WindowN obsNotes
post2010_ex_covid~156Long baseline; captures full ISM history (1959–2016 FRED-MD + 2021–2025 DB.nomics)
post2018_ex_covid~60Post oil-price run-up; avoids 2015–2018 quiet zone
post2023_full36Captures Houthi + tariff structural shifts; ISM real data available throughout
post2023_ex_covid36Same window; COVID months formally excluded (no effect since training starts 2023)

post2015 window dropped: The 2015–2019 quiet period (GSCPI ≈ 0, elevated freight) taught inverted β for freight and fuel blocks. April 2026 attribution would have been severely mis-specified.

KNOWN ISSUES & LIMITATIONS
⚠ Data Gaps

ISM 2016–2020 gap: NAPMPI and NAPMSDI dropped by FRED after April 2016. Real data restored: FRED-MD vintage 1959–2016, DB.nomics ISM 2021–2025, ISM snapshot 2026. Gap 2016-05–2020-12 (56 months) remains NaN — not freely available; training windows that need it (post2010, post2018) use fewer joint obs.

BTS TSI April 2026: Not yet released (~Jun 5–10). Block mean-imputed to z=0. Estimated lost attribution: ~+0.10 pp.

RWI/ISL April 2026: Not yet released (~Jun 5–10). Negative PC1 loading, so NaN imputation is approximately neutral.

Census M3 full: AMTMNO and AMTMUS NaN for April 2026. Full release ~late June 2026.

⚡ Counter-Cyclical Blocks

RWI/ISL Container (PC1: −0.246): Negative loading expected to be positive. COVID demand surge caused high throughput AND high GSCPI simultaneously — reversing the "fewer containers = more pressure" logic. MoM z-score approach tested but correlation drops to r = +0.047.

LA/LB Port Pressure (PC1: −0.271): Same root cause as RWI/ISL. Both blocks have combined attribution +0.082 pp for April 2026 (partially offsetting).

ISM PI vs Philly/Empire (r = 0.9+): Ridge splits price-pressure coefficients unstably across ISM, Philly, Empire. ISM PI carries negative coefficient in post2023 model despite positive z-score; combined price attribution is stable at ~+0.35 pp.

✓ Resolved Issues

post2015 model dropped: Inverted β in 2015–2019 quiet period. Attribution improved from 56% → 58% after dropping.

ISM real data restored: Regression imputation replaced by real DB.nomics ISM data for 2021–2025. PCA r improved from 0.69 → 0.75. Real COVID-era values (SDI 68–78, PI 82–92 in 2021) now correctly anchor the series.

PortWatch Bosporus/Cape: Added as reference blocks (r_post2023 = +0.607, +0.422). NOT in attribution model — pre-event noise in longer windows degrades post2018 β estimates.
