# Simon Fuchs — research website

Source for [sfuchs-de.github.io](https://sfuchs-de.github.io/), the research website of Simon Fuchs.

The site presents current working papers, publications, policy work, work in progress, and a downloadable CV. It is a lightweight static site hosted with GitHub Pages.

## Structure

- `index.html` — homepage and research list
- `css/main.css` — responsive site styles
- `Pictures/CV.pdf` — current CV
- `research/` — paper and presentation files

Research content and views are Simon Fuchs's own and do not necessarily reflect those of the Federal Reserve Bank of Atlanta or the Federal Reserve System.
